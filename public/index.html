Image search application

User stories:
  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=10 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example usage:

You may query text ('q'), latitude ('lat') and longitude ('lon').

You may paginate through the results by adding 'offset' parameter. Max value is 90.

https://sergejuss-image-search.herokuapp.com/search?q=vilnius

https://sergejuss-image-search.herokuapp.com/search?q=gaudi&lat=41.40348&lon=2.17442

https://sergejuss-image-search.herokuapp.com/search?q=gaudi&lat=41.40348&lon=2.17442&offset=10
